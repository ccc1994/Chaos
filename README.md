# AI Coding Agent

This project is a Python-based AI coding agent designed to automate coding tasks.

## Features
- AI-driven code generation and modification.
- Integration with popular LLMs.
- Extensible tool system.

## Getting Started

### Prerequisites
- Python 3.9 or higher

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -e .
   ```

### Usage
Run the main script:
```bash
python src/main.py
```
